Cross-Sim

CrossSim is a crossbar simulator designed to model resistive memory crossbars for both neuromorphic computing and digital memories. It provides a clean python API so that different algorithms can be built upon crossbars while modelling realistic device properties and variability.

Three operations:

Vector *Matrix: .run_xbar_vmm

Matrix*Vector: .run_xbar_mvm

Vector*Vector: .update_matrix
